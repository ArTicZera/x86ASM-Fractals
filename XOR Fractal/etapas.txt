//ALGORITMO PARA CONFIGURAR O PROGRAMA
1 - ESPECIFICAR 16 BITS
2 - CONFIGURAR O PROGRAMA PARA RODAR NO AMBIENTE DE INICIALIZAÇÃO
3 - USAR O MODE 13H (VGA; 256 CORES; 320x200)
4 - CONFIGURAR O SEGMENTO DE VÍDEO (0xA000)
5 - CONFIGURAR OS REGISTRADORES (AH, AL, BX, CX, DX)
6 - PREENCHER O SETOR DE BOOT COM 510 BYTES COM 0 & 2 BYTES --> 0xAA55

//ALGORITMO PARA GERAÇÃO DE PIXELS
1 - ITERAR CADA PIXEL PARA PLOTTAR NA TELA (320x200)
2 - CALCULAR CADA PIXEL COM UM "PADRÃO" (EX: X XOR Y)

//ALGORITMO PARA GERAÇÃO DE CORES VIVAS
SE AL FOR MAIOR QUE 55 --> SUBTRAIA 16
SE AL FOR MENOR QUE 32 --> ADICIONE 32
SE AL ESTIVER ENTRE 55 AND 32 --> DEFINE PIXEL

//BONUS
1 - CRIE UMA VARIÁVEL DO TIPO "INTEIRO" QUE COMEÇE COM 0
2 - ADICIONE A VARIÁVEL PARA O REGISTRADOR AL DEPOIS DE CALCULAR O "PADRÃO"

QUANDO A TELA ESTIVER TOTALMENTE PREENCHIDA COM PIXELS, REINICIE CADA COORDENADA
E COMEÇE TUDO DE NOVO MAS COM A VARIÁVEL INCREMENTADA.
